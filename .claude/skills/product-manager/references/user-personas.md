**引述**：
> 「我不是很相信命運之類的，但有時候真的很迷惘，需要一個方向感。
> 塔羅感覺比心理諮商輕鬆多了，不用那麼正式。」

## 使用者旅程地圖

### 完整生命週期

```
Awareness → Trial → Habit → Premium → Advocate
  (認知)     (試用)   (習慣)    (付費)     (推薦)
```

### 階段 1: Awareness（認知）

| 觸點 | 內容 | 成功指標 |
|------|------|---------|
| IG/TikTok 短影音 | 每日塔羅運勢、心理測驗 | 觀看數、互動率 |
| Threads 貼文 | 靈性短詩、塔羅小知識 | 分享數、點擊率 |
| SEO Blog 文章 | 「塔羅初學者指南」「水晶功效大全」 | 自然搜尋流量 |
| KOL 推薦 | 塔羅/星座 KOL 使用體驗分享 | 推薦碼安裝量 |

**轉化指標**：Visitor → Signup = 40%

### 階段 2: Trial（試用）

| 步驟 | 體驗 | 關鍵時刻 |
|------|------|---------|
| 1. 註冊 | LINE Login 一鍵登入 | 零摩擦 |
| 2. 命理建檔 | 輸入出生資料，看到星座/五行/生肖 | 「哇，好準！」驚喜感 |
| 3. 首次解讀 | 選主題 → 選牌 → AI 解讀 | 核心 Aha Moment |
| 4. 結果頁 | 解讀內容 + 行動建議 + 水晶推薦 | 價值感知 |

**轉化指標**：Signup → First Reading = 70%

### 階段 3: Habit（習慣）

| 觸發 | 行為 | 獎勵 |
|------|------|------|
| 情緒低落 | 打開 App 做解讀 | 感覺被理解 |
| 日常好奇 | 每日運勢查看 | 方向感 |
| LINE 關懷訊息 | 回到 App | 感覺被記住 |
| 記憶回顧 | 看到「上次你提到...」 | 連結感加深 |

**轉化指標**：First Reading → D7 Return = 28%+

### 階段 4: Premium（付費）

| 觸發 | 阻礙 | 解決方案 |
|------|------|---------|
| 每日 3 次額度用完 | 「NT$299 值得嗎？」 | 顯示已節省的傳統塔羅費用 |
| 想要更深入解讀 | 價格敏感 | 首月半價 NT$149 試用 |
| 想要主動關懷 | 不確定效果 | 展示範例關懷訊息 |
| 朋友推薦 | 信任問題 | 推薦碼雙方優惠 |

**轉化指標**：Active User → Premium = 4%+

### 階段 5: Advocate（推薦）

| 觸發 | 行為 | 強化 |
|------|------|------|
| 精準解讀 | 截圖分享到 IG Story | 提供精美分享卡模板 |
| 朋友聊天提到 | 口頭推薦 | 推薦碼機制 |
| 社群互動 | 在 FB 社團/Threads 分享體驗 | 社群互動活動 |
| KOL 合作 | 發布使用心得影片 | KOL 專屬推薦碼 |

**轉化指標**：Premium → Advocate = 10%

## 痛點 vs 增益 對照表

### 痛點 (Pains)

| 痛點 | 嚴重度 | 受影響 Persona | 現有替代方案的不足 |
|------|--------|---------------|------------------|
| 塔羅太貴 | 高 | 小雨、美玲 | 傳統塔羅 NT$500~2,000/次 |
| 千篇一律的回覆 | 高 | 小雨、Jenny | 占卜 App 制式化 |
| 沒人記得我說過什麼 | 高 | 美玲 | 所有現有方案都無記憶 |
| 想找人聊但怕被批判 | 中 | 美玲、小雨 | 真人塔羅師可能批判 |
| 沒有主動關心我 | 中 | 美玲 | 所有現有方案都是被動 |
| 台灣沒有好用的命理 App | 中 | Jenny | Co-Star 無中文 |
| 解讀品質不一致 | 中 | 阿凱 | 不同塔羅師品質差異大 |
| 想要但覺得「迷信」 | 低 | 阿凱 | 塔羅仍有社會壓力 |

### 增益 (Gains)

| 增益 | 產品功能 | 價值主張 |
|------|---------|---------|
| 隨時隨地、低成本解讀 | AI 塔羅 + Premium | 「一天 NT$10，24/7 塔羅夥伴」 |
| 每次解讀都獨一無二 | AI 個人化 + 記憶 | 「它真的認識我」 |
| 有人記得我的故事 | 三層記憶系統 | 「跨越數月，它記得我說過的每句話」 |
| 情緒低落時有人關心 | 主動關懷引擎 | 「不用開口求助，它知道我需要」 |
| 精美的社交分享 | 分享卡片 + UI 設計 | 「好看到想分享到 IG」 |
| 水晶推薦的個人化 | 五行×情緒×星座 | 「這顆水晶就是為我選的」 |
| 科學感的命理整合 | 多源數據融合 | 「不只是塔羅，是全方位的自我了解」 |
