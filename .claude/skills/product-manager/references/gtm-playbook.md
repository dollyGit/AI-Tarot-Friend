### Phase A: 預備期 (Month 0~3)

**目標**：建立品牌存在感 + 累積初始社群 + 完成核心產品

| 維度 | 計劃 |
|------|------|
| **MAU 目標** | 0（產品未上線） |
| **社群目標** | 500 followers (IG + Threads + TikTok 合計) |
| **月度支出** | ~NT$600K/月（主要是工程團隊） |
| **營收** | NT$0 |

#### 工程進度（對應 WBS P0~P4）

| 月份 | 里程碑 | 產出 |
|------|--------|------|
| M1 | P0 Scaffolding + P1 DAL Start | Docker Compose、Protobuf、Service Template |
| M2 | P1 DAL Complete + P2 Customer Core | gRPC CRUD、Cache、Customer API |
| M3 | P3 Advanced Engines + P4 Caring | MongoDB、InfluxDB、Qdrant、Neo4j、Caring v1 |

#### 社群內容策略（M1 開始）

| 內容類型 | 頻率 | 平台 | 目的 |
|---------|------|------|------|
| 每日塔羅運勢 | 1~2 videos/day | TikTok + IG Reels | 吸引塔羅愛好者 |
| AI 手相解讀 | 1 video/day | IG Reels + Stories | 好奇心引流 |
| 心理測驗 | 2~3 posts/day | Threads + IG | 互動 + 分享 |
| 靈性短詩 | 2~3/day | Threads + IG Stories | 情感連結 |
| 水晶小知識 | 3~5/week | All channels | 教育 + 鋪墊電商 |
| SEO 長尾文章 | 2~3 articles/week | Blog | 自然搜尋流量 |

**內容生產方式**：AI 自動生成 + 人工審核，每日產出成本極低

#### 品牌建設

- 確定品牌視覺語言（神秘但不迷信、現代但有靈性）
- 建立 IG/Threads/TikTok/LINE OA 帳號
- 設計品牌 CI（Logo、色彩、字體、Social Media templates）
- 撰寫品牌故事 & About Us

## 漏斗指標

### 完整漏斗追蹤

```
Stage 1: Awareness (觸及)
  ├── Social Media Impressions
  ├── SEO Organic Traffic
  ├── KOL Referrals
  └── Paid Ad Impressions
       │
       ▼ CTR: 2~5%
Stage 2: Visit (造訪)
  ├── Landing Page Views
  └── App Store Views
       │
       ▼ Signup Rate: 40%
Stage 3: Signup (註冊)
  ├── LINE Login 完成
  └── 命理建檔完成 (80%)
       │
       ▼ First Reading: 70%
Stage 4: Activation (啟用)
  ├── 完成首次塔羅解讀
  └── 解讀滿意度 ≥ 4.0
       │
       ▼ D7 Return: 28%
Stage 5: Retention (留存)
  ├── 每週 2+ 次解讀
  └── 主動關懷回應
       │
       ▼ Conversion: 4%+
Stage 6: Revenue (營收)
  ├── Premium 訂閱
  ├── 水晶購買
  └── 塔羅師預約
       │
       ▼ Referral: 10%
Stage 7: Advocacy (推薦)
  ├── 分享到社群
  ├── 推薦碼使用
  └── App Store 5★ 評價
```

### 各階段轉化率基準

| 轉化 | M4 (上線) | M6 (成長) | M9 (PMF) | M12 (規模) |
|------|----------|----------|----------|-----------|
| Visit → Signup | 30% | 35% | 40% | 40% |
| Signup → First Reading | 60% | 65% | 70% | 75% |
| First Reading → D7 | 20% | 24% | 28% | 30% |
| Active → Premium | 3% | 5% | 8% | 10% |
| Premium → Advocate | 5% | 8% | 10% | 12% |

## KOL / 內容策略

### KOL 合作類型

| 類型 | 粉絲量 | 合作方式 | 預算 | 預期效果 |
|------|--------|---------|------|---------|
| **Nano (種子)** | 1K~5K | 免費試用 | NT$0 | 真實口碑 |
| **Micro (核心)** | 5K~50K | 試用 + 推薦碼 | NT$3~10K/位 | 精準獲客 |
| **Mid (擴展)** | 50K~200K | 現金 + 推薦碼 | NT$15~50K/位 | 品牌曝光 |
| **Macro (爆破)** | 200K+ | 整合行銷 | NT$80~200K/位 | 大規模觸及 |

### 目標 KOL 類型

| 類型 | 代表性帳號特徵 | 合作重點 |
|------|--------------|---------|
| 塔羅 YouTuber | 每週塔羅運勢影片 | 深度評測 + 教學影片 |
| 星座 KOL | IG/Threads 星座內容 | 日常分享 + Story 合作 |
| 心靈成長 Podcast | 訪談 + 自我探索主題 | Podcast 專訪 + 推薦碼 |
| 水晶收藏家 | 水晶開箱 + 功效介紹 | 聯名水晶推薦 |
| 生活風格 Blogger | Self-care + 日常 routine | 產品置入日常 |

### 合作流程

```
1. KOL 篩選（互動率 > 3%、調性匹配、粉絲受眾吻合）
2. 初次接觸（DM or Email，附品牌 kit）
3. 試用邀請（Premium 免費 3 個月 + 專屬推薦碼）
4. 內容協調（不限制創作，但提供品牌重點）
5. 發布追蹤（UTM + 推薦碼 + 下載追蹤）
6. 效果評估（CPA、安裝數、轉化率）
7. 長期合作（表現好的 KOL 簽訂季度/年度合約）
```

## 成長飛輪

### 核心飛輪

```
         ┌──── 更多用戶數據 ◄────┐
         │                       │
         ▼                       │
    Memory 品質提升         新用戶獲取
         │              (CAC 降低)
         ▼                       │
    Retention 提高              │
    (用戶黏性增強)                │
         │                       │
         ▼                       │
    口碑傳播增加 ──────────────────┘
    (NPS ↑, 分享 ↑)
```

### 飛輪加速器

| 加速器 | 機制 | 效果 |
|--------|------|------|
| **記憶深度** | 使用越久，AI 越了解你 | 遷移成本 ↑，Retention ↑ |
| **關懷精準度** | 數據越多，關懷越精準 | 用戶滿意度 ↑，NPS ↑ |
| **推薦準確度** | 購買數據改善推薦 | 水晶轉化率 ↑，Revenue ↑ |
| **內容品質** | 用戶故事豐富內容素材 | 社群內容品質 ↑，Acquisition ↑ |
| **KOL 社會證明** | 真實用戶成為 KOL | 信任度 ↑，轉化率 ↑ |

### 飛輪量化指標

| 飛輪階段 | 指標 | M6 | M9 | M12 |
|---------|------|-----|-----|------|
| Memory 品質 | 平均記憶深度（層數） | 1.2 | 2.0 | 2.5 |
| Retention | D7 | 24% | 28% | 30% |
| 口碑 | NPS | 35 | 45 | 50+ |
| Acquisition | Organic % | 95% | 80% | 60% |
| Revenue | ARPU | NT$200 | NT$350 | NT$450 |

## 風險管理

### GTM 風險矩陣

| 風險 | 可能性 | 影響 | 緩解策略 |
|------|--------|------|---------|
| AI 解讀品質不穩定 | 中 | 高 | 嚴格 QA + 快速修復循環 |
| 付費轉化率低於預期 | 中 | 高 | A/B 測試定價 + 價值溝通 |
| KOL 合作效果不佳 | 中 | 中 | 分散投資 + 嚴格 CPA 追蹤 |
| 競品抄襲 | 低 | 中 | 加速記憶飛輪 + 專利保護 |
| 通用 AI 蠶食市場 | 高 | 中 | 強化差異化（記憶+關懷+電商） |
| 台灣市場太小 | 低 | 高 | 驗證 PMF 後進入香港/東南亞 |
| 資金不足 | 低 | 高 | 嚴格控制 burn rate + M9 break-even |

### 應急計劃

| 情境 | 觸發條件 | 行動 |
|------|---------|------|
| 用戶增長停滯 | MAU 連續 2 月無增長 | 重新評估頻道策略 + 加大內容投資 |
| 轉化率過低 | Premium 轉化 < 2% | A/B 測試定價 + 增加試用期 + 改善 onboarding |
| 高 churn | 月流失率 > 15% | 深度用戶訪談 + 加強主動關懷 |
| 資金緊張 | Runway < 6 個月 | 削減非核心支出 + 加速營收 + 啟動融資 |
| 技術事故 | 重大停機 > 4 小時 | 事後檢討 + 加強監控 + SLA 改善 |
