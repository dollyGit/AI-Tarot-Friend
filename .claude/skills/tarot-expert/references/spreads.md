# Spread Layouts — Detailed Position Meanings & Reading Guide

## 1-Card Spread — Daily Insight

### Position Meanings

```json
[
  {
    "position": 1,
    "meaning_en": "Core message — the energy or theme most relevant to you right now",
    "meaning_zh": "核心訊息 — 此刻與你最相關的能量或主題"
  }
]
```

### Reading Guide

- Focus on the full spectrum of the card's meaning
- Consider both upright and reversed as nuances, not opposites
- Connect to the user's stated question or general life context
- Keep interpretation concise but meaningful (3-5 sentences)
- Best for: daily guidance, quick check-ins, yes/no energy readings

## 3-Card Spread — Time-Based

### Layout A: Past → Present → Future (Default)

```json
[
  {
    "position": 1,

    "position": 2,

    "meaning_zh": "現在 — 目前的能量，你正在經歷的"

    "position": 3,
    "meaning_en": "Future — where things are heading if the current path continues",
    "meaning_zh": "未來 — 如果維持目前方向，事情的走向"

]
```


### Layout C: Mind → Body → Spirit

```json
[
  {
    "position": 1,
    "meaning_en": "Mind — your mental state and thought patterns",
    "meaning_zh": "心智 — 你的心理狀態與思考模式"
  },
  {
    "position": 2,
    "meaning_en": "Body — physical health, material concerns, actions",
    "meaning_zh": "身體 — 身體健康、物質層面、行動"
  },
  {
    "position": 3,
    "meaning_en": "Spirit — spiritual growth, intuition, deeper purpose",
    "meaning_zh": "靈性 — 靈性成長、直覺、更深層的目的"
  }

```

### Reading Guide

- Read the three cards as a story arc: beginning → middle → resolution
- Note elemental interactions between adjacent cards
- The middle card often holds the key insight
- If all three are the same suit: strong focus on that life area
- Best for: focused questions about specific situations

## 7-Card Horseshoe Spread

### Layout

```
Position:  [1]  [2]  [3]  [4]  [5]  [6]  [7]
```

### Position Meanings

```json
[
  {
    "position": 1,
    "meaning_en": "Past — events and energies from the past shaping the current situation",
    "meaning_zh": "過去 — 塑造目前狀況的過往事件與能量"
  },
  {
    "position": 2,
    "meaning_en": "Present — your current state and circumstances",
    "meaning_zh": "現在 — 你目前的狀態與處境"
  },
  {
    "position": 3,
    "meaning_en": "Hidden influences — what you're not seeing, subconscious factors",
    "meaning_zh": "隱藏的影響 — 你未察覺的、潛意識的因素"
  },
  {
    "position": 4,
    "meaning_en": "Obstacles — what's standing in your way",
    "meaning_zh": "障礙 — 阻擋你前進的因素"
  },
  {
    "position": 5,
    "meaning_en": "External influences — people and environment affecting you",
    "meaning_zh": "外在影響 — 影響你的人物與環境"
  },
  {
    "position": 6,
    "meaning_en": "Advice — the best approach to take",
    "meaning_zh": "建議 — 最佳的應對方式"
  },
  {
    "position": 7,
    "meaning_en": "Probable outcome — where this is heading",
    "meaning_zh": "可能的結果 — 事情的走向"
  }
]
```

### Reading Guide

- Read positions 1-2 for context (where you've been and where you are)
- Position 3 is often the most insightful — what's hidden matters most
- Positions 4-5 show what the user needs to navigate
- Position 6 is the actionable advice
- Position 7 is NOT fixed destiny — it's the likely outcome if current energy continues
- Elemental dignity: check interactions between adjacent pairs (1-2, 2-3, 3-4, etc.)

### Layout

```
            [3]  Crown
        [5] [1/2] [6]
            [4]  Foundation
                  [10] Outcome
                  [9]  Hopes/Fears
                  [8]  Environment
                  [7]  Self
```

### Reading Order & Narrative Construction

**Phase 1 — The Central Cross (Positions 1-6)**
Read as a story: "You are here (1), challenged by (2), aiming for (3), driven by (4), coming from (5), moving toward (6)."

**Phase 2 — The Staff (Positions 7-10)**
Read as deeper context: "You see yourself as (7), others see you as (8), you hope/fear (9), leading to (10)."

**Phase 3 — Synthesis**
Connect: How does the central story (1-6) interact with the personal context (7-10)? Look for:
- Cards that echo each other (same number, same suit, same element)
- The relationship between position 3 (goal) and position 10 (outcome) — are they aligned?
- Position 9 (hopes/fears) often reveals what the user needs to hear most
- Elemental dignity between positions 1 and 2 is especially significant
